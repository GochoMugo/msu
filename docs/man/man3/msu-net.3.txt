MSU-NET(3)
==========
:doctype: manpage


NAME
----
msu-net - common network operations.


SYNOPSIS
--------
Simplifying network operations for the user.

    msu_require "net"


DESCRIPTION
-----------
*track('LABEL', 'URL')*::
    Track the download with the label 'LABEL' and URL 'URL'. by saving it
    to a file named '.msu.download' in the current working directory.

*check(['LABEL'...])*::
    Checks if the downloads with the labels 'LABEL' are tracked.

    Alias: 'net.ch'

*download('LABEL', 'URL')*::
    Downloads an asset at URL 'URL' and tracking it using the label 'LABEL'.
    Note that the file is not saved with the name 'LABEL'.

    Alias: 'net.dl'
